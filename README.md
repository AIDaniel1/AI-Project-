# AI-Project-
Designing of an AI RAG (retriever augmented generation).


AI Agent Pipeline with Vector Database

A complete AI agent pipeline featuring a vector database for knowledge retrieval, LLM integration for intelligent responses, and a web interface for user interaction. Built with FastAPI, ChromaDB, and Streamlit.


 Features

路  Vector Database: ChromaDB for efficient similarity search and knowledge retrieval
路  LLM Integration: OpenAI GPT support with fallback to mock responses
路  RAG Pipeline: Retrieval-Augmented Generation for context-aware responses
路  Conversation Memory: Maintains chat history for contextual conversations
路  Web Interface: Streamlit-based UI for easy interaction
路  REST API: FastAPI backend with comprehensive endpoints
路  Knowledge Management: Add, search, and manage documents in real-time


 Installation

Prerequisites

路 Python 3.8+
路 OpenAI API Key (optional - Not available for the project)
